# Caesar-Cipher-Encode-and-Decode
This is a Caesar Cipher Encode/Decode project if you want to encode or decode any word then you can ues ths code if you want, Example: Given the string "HELLO" and a shift value of 3, the encoded result will be "KHOOR". Decoding "KHOOR" with a shift value of 3 will return the original "HELLO".
